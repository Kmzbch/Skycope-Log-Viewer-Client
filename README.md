# SkycopeLogViewerClient

This is a front-end solution project to the mini-project tech assessment of Skycope Technologies.

## How to run

$ npm install
$ npm start

Navigate to http://localhost:4200/

## What are covered
## Covered
- Python 3
    - Flask
- TypeScript
    - Angular12
- HTML 5
- Bootstrap
- CSS3/SASS/SCSS
- RESTful API

## Not covered

- NGINX
- Deloployment on cloud(e.g. AWS)
